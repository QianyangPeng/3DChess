运行说明：

本程序运行环境为python 2.7

依赖库有pygame-1.9.1、PyOpenGL-3.1.0、numpy-1.6.2与OpenGLLibrary，其中pygame-1.9.1、PyOpenGL-3.1.0和numpy-1.6.2的安装包都放置在“支持库”文件夹中，在装有python2.7的电脑上直接双击安装即可。OpenGLLibrary无需安装，直接保留在游戏根目录中即可正常调用。
所有文件或文件夹请勿修改名字或移动删除，否则游戏将无法正常运行。

――――――――――――――――――――――――――――――――
游戏说明：

鼠标左键：选中棋子，放下棋子，点击按钮

键盘方向键：调整视角（仅在下棋界面有效）

Esc键：退出游戏
